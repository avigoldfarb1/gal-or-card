התקנה והפעלה ראשונית

שלב-אחר-שלב

⚠️

דרישות מערכת מומלץ Windows 10/11 (64 ביט). גודל גופן Windows חייב להיות 100% – גופן 125% מעוות תמונות.

  • 1

    קבלת קובץ ההתקנה

    גל אור ישלחו תקליטור CD/DVD או קובץ .exe. לחלופין הורידו Project Viewer מאתר Cybermed.

  • 2

    הפעלת Setup.exe

    לחצו פעמיים על Setup.exe ← עקבו אחר האשף [Next]. בחרו Project Viewer לצפייה בלבד.

    💡 לשינוי שפה בחרו [Show Language Selection Dialog] בזמן ההתקנה.

  • 3

    הפעלת רישיון

    בפתיחה ראשונה לחצו [Register]. לאחר פקיעת הניסיון (30 יום) – חלון [Activation] יופיע. הכניסו מפתח סדרתי ← [Register].

    ⚠️ נדרש חיבור אינטרנט לרישום.

  • 4

    הגדרת ברירת מחדל

    לחצו פעמיים על הרישיון ← [Set as Default]. מקצר זמן טעינה. נדרשת הפעלה מחדש.

  • 5

    הגדרת תצורה

    Start → All Programs → OnDemand3D → [Configure OnDemand3D]. הגדרת DB, DICOM, DBM, MPR ועוד.

ממשק המשתמש

אזורי העבודה המרכזיים

🗂️ מנהל מסד הנתונים (DBM)

ה-DBM הוא שער הכניסה לתוכנה. ממיינים נתוני DICOM של מטופלים, קבצי פרויקט, דוחות וקבצי mesh. שלוש נקודות גישה:

  • Master DB – בסיס הנתונים המקומי
  • OnDemand3D Server – שרת מרכזי עם גישה מרחוק
  • Remote PACS Server – חיבור דרך פרוטוקול DICOM
🔍

Database Explorer

חפשו מטופל, תאריך, סוג סריקה. ניתן להעתיק נתונים ממערכת למערכת.

⚙️

Quick List ו-Toolbar

הגדירו מיקום ב-[Toolbar Position]. כפתורי גישה מהירה חוסכים זמן יקר.

💾

ייצוא לתקליטור/USB

כתיבה ישירה ל-CD/DVD/USB/רשת. DVD מומלץ לנפחים גדולים.

🖼️

X-Image – ניהול משולב

נתוני 2D ו-3D בפריסה משולבת. ממיר קבצי תמונה ל-DICOM לשידור ל-PACS.

קבלת CD / USB מגל אור

2

פתיחת נתוני DICOM ב-DBM

DBM → [Data Source] → אתרו תיקיית DICOM → לחיצה כפולה על תיקיית המטופל.

3

סקירה ב-Dynamic Light Box

גיללו בין פרוסות Axial/Sagittal/Coronal. 3D Zoom לסקירה ראשונה. Virtual Endoscopy לניווט בסינוסים.

4

ניתוח במודול Dental

פנורמה, חתכים, TMJ בילטרלי. Preference: סמן עצב, Safety Cylinder.

5

תכנון שתל ב-In2Guide

בחרו שתל מהספרייה, מקמו, עומק וזווית. הזמינו מדריך כירורגי.

6

יצירת דוח ושמירה

X-Report / Report לתיעוד ודוח HTML. שמרו Project File בשרת לגישה ממרפאה אחרת.

8

עבודה עם DICOM

ייבוא, ייצוא, שרתים ותאימות

📁 פורמט DICOM – תקן בינלאומי

DICOM 3.0 הוא התקן הבינלאומי לתמונות רפואיות. OnDemand3D תומכת במלואו – multi-frame, split-frame, והמרה ביניהם. כל שידורי הנתונים מוצפנים.

  • A

    ייבוא נתונים

    DBM → [Data Source] → אתרו תיקיית DICOM → גיררו לחלון ה-DBM.

  • B

    חיבור לשרת OnDemand3D

    [Server] ב-Data Source → שם משתמש, סיסמה, כתובת שרת → [Connect].

    💡 Project Files בשרת נגישים מכל מחשב עם אינטרנט.

  • C

    חיבור ל-PACS

    חיבור דרך DICOM לכל מערכת PACS. שגרו תמונות שנלכדו ישירות מ-Report Module.

  • D

    כתיבה לתקליטור/USB

    DBM → נתון → [Write CD]. קבצי ספריית שתלים מגדילים נפח – מומלץ DVD.

תכנון שתלים ו-In2Guide

מהסריקה לכירורגיה מודרכת

🦴 ספריית שתלים עולמית

ספריית שתלים אמיתיים ממגוון יצרנים מובילים בעולם. תכנון וירטואלי עם מידות מדויקות, שיניים וירטואליות והפקת מדריך כירורגי.

  • 1

    פתיחת In2Guide

    DBM → פעמיים על הסריקה → [Open in In2Guide / Dental].

  • 2

    בחירת שתל מהספרייה

    [Implant Library] → יצרן, דגם, רוחב, אורך.

    💡 ניתן להוסיף ספריות בהתאמה אישית.

  • 3

    מיקום, עומק וזווית

    גיררו חץ הכיוון. Preference → [Cylinder Margin (mm)].

    ⚠️ חץ הכיוון אינו ניתן להסרה – הוא הכלי לכיוון השתל.

  • 4

    הזמנת מדריך כירורגי

    הזמינו מדריך מותאם אישית ישירות מ-In2Guide.
