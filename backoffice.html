🔑 GitHub Token
📖 הוראות יצירת Token
  1. היכנס ל-github.com עם חשבונך
  2. לחץ על תמונת הפרופיל (פינה ימנית עליונה) ← Settings
  3. גלול למטה עד Developer settings (בתפריט הצד)
  4. בחר Personal access tokens ← Tokens (classic)
  5. לחץ Generate new token (classic)
  6. תן שם לטוקן: gal-or-backoffice
  7. סמן את repo (כולל כל תתי-הרשאות)
  8. הגדר תפוגה (Expiration) לפי רצונך
  9. לחץ Generate token — העתק מיד (לא יוצג שוב!)
  10. הדבק בשדה Token למעלה ולחץ שמור הגדרות
