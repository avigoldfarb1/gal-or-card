🔘ניהול כפתורי הפעולה
ניתן להפעיל/לכבות כפתורים ולערוך את הטקסט שלהם בעברית ואנגלית.
🐙הגדרות GitHub — פרסום אוטומטי
🔑 איך ליצור Personal Access Token (פעם אחת בלבד)
  1. היכנס ל-github.com → לחץ על תמונת הפרופיל שלך (למעלה מימין)
  2. לחץ Settings → גלול למטה ← לחץ Developer settings
  3. לחץ Personal access tokens → Tokens (classic)
  4. לחץ Generate new token (classic)
  5. תן שם לטוקן: backoffice-card
  6. תחת Select scopes — סמן ✅ repo (הראשון ברשימה)
  7. גלול למטה ← לחץ Generate token
  8. העתק את הטוקן (מתחיל ב-ghp_) והדבק למטה
⚠️ הטוקן נשמר רק בדפדפן שלך (localStorage) ולא נשלח לשום מקום חיצוני.
🚀פרסום ישיר ל-GitHub Pages
לאחר שמירת ה-Token, לחץ "פרסם לאוויר" — הכרטיס יתעדכן תוך 30-60 שניות.
📋יומן פרסום
🚀 פרסום לאוויר
💾ייצוא קוד HTML
לחץ "ייצא HTML" כדי להוריד קובץ index.html מעודכן עם כל השינויים שביצעת. העלה את הקובץ ל-GitHub Pages (או לכל שרת אחר) כדי לעדכן את הכרטיס החי.
📋 הוראות פרסום ב-GitHub Pages:
  1. לחץ "ייצא HTML" להורדת הקובץ
  2. פתח את ה-Repository שלך ב-GitHub
  3. לחץ "Add file" → "Upload files"
  4. גרור את הקובץ index.html
  5. לחץ "Commit changes"
  6. הכרטיס יתעדכן תוך 1-2 דקות
